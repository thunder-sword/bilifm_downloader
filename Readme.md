# 0x1.概述

可以将b站音频下载下来，并转换为mp3格式的图形化程序（长辈下载音乐给车机用）。

依赖bilifm项目，同时本地还需要安装ffmpeg。

自用。

# 0x2.版本

## v0.3.0

支持下载更多种类的视频的音频（不包括AV号）：

| 类型 | 示例         | 含义                    |
| ---- | ------------ | ----------------------- |
| BV号 | BV1xx411c7mD | 视频稿件ID              |
| AV号 | av170001     | 视频稿件ID（旧）        |
| EP号 | ep363688     | 剧集ID                  |
| SS号 | ss43018      | 番剧季度ID（season_id） |
| MD号 | md28231846   | 媒体ID（media_id）      |

## v0.2.0

修复一个BV号存在多个视频的问题；

处理续下的问题，跳过下载和转换过的音频，并跳过下载只转化还没转化的视频；

## v0.1.0

基础功能完成，可下载视频并转化成mp3音频。
